# Product

Pharos AI Runtime

# Purpose

Pharos AI Runtime is the execution engine used by AI employees inside Pharos.

It is responsible for loading configuration, resolving agents, executing workflows, and coordinating internal operations.

# Scope

Current scope:

- Runtime
- Agent execution
- Workflow execution
- Internal tools
- Internal memory interfaces

Not in scope:

- OpenAI
- Claude
- Gemini
- External APIs
- HTTP clients
- Firestore
- Background jobs
- Token consumption

# Principles

- Simple
- Deterministic
- Testable
- Modular
- Dependency Injection
- Offline First

# Long-Term Vision

Become the common execution runtime for every AI employee used across Pharos products.
